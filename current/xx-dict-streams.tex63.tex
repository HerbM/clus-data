====== Condition Type STREAM-ERROR ======

====Class Precedence List==== **[[CL:Types:stream-error]]**, **[[CL:Types:error]]**, **[[CL:Types:serious-condition]]**, **[[CL:Types:condition]]**, **[[CL:Types:t]]**

====Description====

The type **[[CL:Types:stream-error]]** consists of error conditions that are related to receiving input from or sending output to a //[[CL:Glossary:stream]]//. The "offending stream" is initialized by \theinitkeyarg{stream} to **[[CL:Functions:make-condition]]**, and is //[[CL:Glossary:accessed]]// by the function **[[CL:Functions:stream-error-stream]]**.
