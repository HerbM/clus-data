====Compound Type Specifier Kind====
Abbreviating.

====Compound Type Specifier Syntax====
  * //**#1** [lower-limit [upper-limit]]//

====Compound Type Specifier Arguments====
  * //lower-limit//, //upper-limit// - //[[CL:Glossary:interval designator|interval designators]]// for //[[CL:Glossary:type]]// **[[CL:Types:#1]]**. The defaults for each of //lower-limit// and //upper-limit// is the //[[CL:Glossary:symbol]]// **[[CL:Types:wildcard|*]]**.

====Compound Type Specifier Description====
This denotes the //[[CL:Glossary:#1s]]// on the interval described by //lower-limit// and //upper-limit//.
