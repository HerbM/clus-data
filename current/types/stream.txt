====== System Class STREAM ======

====Class Precedence List==== 
**stream**, **[[CL:Types:t]]**

====Description====
A //[[CL:Glossary:stream]]// is an //[[CL:Glossary:object]]// that can be used with an input or output function to identify an appropriate source or sink of //[[CL:Glossary:character|characters]]// or //[[CL:Glossary:bytes]]// for that operation.

For more complete information, see section {\secref\StreamConcepts}.
