====== Function CLASS-OF ======

====Syntax====

====Arguments and Values====
  * //object// - an //[[CL:Glossary:object]]//.

====Description====
Returns the //[[CL:Glossary:class]]// of which the //object// is a //[[CL:Glossary:direct instance]]//.
