====== Condition Type FLOATING-POINT-OVERFLOW ======

**floating-point-overflow**, **[[CL:Types:arithmetic-error]]**, **[[CL:Types:error]]**, **[[CL:Types:serious-condition]]**, **[[CL:Types:condition]]**, **[[CL:Types:t]]**
