====== Type BOOLEAN ======

====Supertypes====

**[[CL:Types:boolean]]**, **[[CL:Types:symbol]]**, **[[CL:Types:t]]**

====Description====

The type **[[CL:Types:boolean]]** contains the //[[CL:Glossary:symbols]]// \misc{t} and \misc{nil}, which represent true and false, respectively.

====See Also====

\misc{t} (//[[CL:Glossary:constant variable]]//), \misc{nil} (//[[CL:Glossary:constant variable]]//), \specref{if}, **[[CL:Functions:not]]**, **[[CL:Functions:complement]]**

====Notes====

Conditional operations, such as \specref{if}, permit the use of //[[CL:Glossary:generalized booleans]]//, not just //[[CL:Glossary:booleans]]//; any //[[CL:Glossary:non-nil]]// value, not just \misc{t}, counts as true for a //[[CL:Glossary:generalized boolean]]//. However, as a matter of convention, the //[[CL:Glossary:symbol]]// \misc{t} is considered the canonical value to use even for a //[[CL:Glossary:generalized boolean]]// when no better choice presents itself.
