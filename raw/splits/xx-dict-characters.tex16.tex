====== Function CHAR-CODE ======

====Syntax====

**char-code** //character// → //code//

====Arguments and Values====

//character// - a //[[CL:Glossary:character]]//.

//code// - a //[[CL:Glossary:character code]]//.

====Description====

**[[CL:Functions:char-code]]** returns the //[[CL:Glossary:code]]// //[[CL:Glossary:attribute]]// of //character//.

====Examples====

<blockquote> ;; An implementation using ASCII character encoding ;; might return these values: (char-code #\\$) → 36 (char-code #\\a) → 97 </blockquote>

====Affected By====

None.

====Exceptional Situations====

Should signal an error of type type-error if //character// is not a //[[CL:Glossary:character]]//.

**[[CL:Functions:char-code-limit]]**

====Notes====

None.
