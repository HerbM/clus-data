====== Variable *PRINT-GENSYM* ======

====Value Type====

a //[[CL:Glossary:generalized boolean]]//.

====Initial Value====

//[[CL:Glossary:true]]//.

====Description====

Controls whether the prefix ""#:'''' is printed before //[[CL:Glossary:apparently uninterned]]// //[[CL:Glossary:symbols]]//. The prefix is printed before such //[[CL:Glossary:symbols]]// if and only if \thevalueof{*print-gensym*} is //[[CL:Glossary:true]]//.

====Examples====

<blockquote> (let ((*print-gensym* nil)) (print (gensym)))
▷ G6040 → #:G6040 </blockquote>

====Affected By====

None.

====See Also====

**[[CL:Functions:write]]**, **[[CL:Variables:*print-escape*]]**

====Notes====

None.
