====== Condition Type FILE-ERROR ======

====Class Precedence List==== **[[CL:Types:file-error]]**, **[[CL:Types:error]]**, **[[CL:Types:serious-condition]]**, **[[CL:Types:condition]]**, **[[CL:Types:t]]**

====Description====

The type **[[CL:Types:file-error]]** consists of error conditions that occur during an attempt to open or close a file, or during some low-level transactions with a file system. The "offending pathname" is initialized by \theinitkeyarg{pathname} to **[[CL:Functions:make-condition]]**, and is //[[CL:Glossary:accessed]]// by the function **[[CL:Functions:file-error-pathname]]**.
